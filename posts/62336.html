1.数组方法

1.遍历元素：

1.会修改原有数组

arr.map((currentValue,index,array) => {} ,this);
--创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。	

arr.forEach((currentValue,index,array) => {} ,this);
--对数组的每个元素执行一次给定的函数。此方法会修改原有数组

使用foreach和map遍历数组的话，使用break不能中断循环，使用return也不能返回到外层函数。

2.不会修改原有数组

arr.join(",");
--数组转字符串，将一个数组的所有元素连接成一个字符串返回

arr.reduce((previousValue,currentValue,index,array) => {},initialValue);
--reduce方法迭代数组的所有项，又称为“累加器”，返回函数最终计算的一个值。

2.查找元素：

1.会修改原有数组

2.不会修改原有数组

arr.filter((element,index,array)=>{},this) ;
--filter方法过滤数组,返回一个新数组, 其保留 通过所提供函数测试的 所有元素。

arr.indexOf(searchElement，fromIndex);
--返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1。

arr.findIndex((element,index,array) => {} ,this);
-- 返回数组中满足提供的测试函数的第一个元素的索引。若没有找到对应元素则返回-1。

arr.find((element,index,array) => {} ,this);
--返回数组中满足提供的测试函数的第一个元素的值。否则返回 undefined。

arr.includes(searchElement，fromIndex); // ES6 
--判断一个数组是否包含一个指定的值，根据情况，返回布尔值

arr.every((element,index,array)=>{},this)
--依据判断条件，数组的元素是否全满足，若满足则返回ture

arr.some((element,index,array)=>{},this)
--依据判断条件，数组的元素是否有一个满足，若有一个满足则返回ture

3.增加元素：

1.会修改原有数组

arr.push(e1, e2, en);
--添加元素到数组的尾部，返回该数组的新长度

arr.unshift(e1, e2, en);
--添加元素到数组的头部，返回该数组的新长度

arr.splice(index, count, e1, e2, en);
--添加元素到数组的任何位置

2.不会修改原有数组

arr.concat(arr1, arr2, arrn) ;
--合并两个或多个数组。

4.删除元素：

1.会修改原有数组

arr.pop();
--删除数组尾部的元素

arr.shift();
--删除数组头部的元素

arr.splice(index, count); 
--删除任意位置元素

delete(arr[2]);
--删除索引值为2的元素值，该位置的元素值改为undefined，返回布尔值

2.不会修改原有数组

arr.slice(start, end);
--包头不包尾的截取数组中的一段，并返回新数组。

5.修改元素：

1.会修改原有数组

arr.fill(value,start,end) ;
--用一个固定值填充一个数组中从起始索引到终止索引内的全部元素。（es6）

6.数组排序：

1.会修改原有数组

arr.sort();
--对数组的元素进行排序，并返回数组

arr.reverse();
--将数组中元素的位置颠倒，并返回该数组

2.备注

1. 数组/对象循环

  1. forEach只能循环数组/对象，不能改变数组里的元素
  2. map循环数组/对象使用return 可以改变数组元素

2. js对象{}、null、[]的区别

在转化布尔值时，对于对象而言，只有null转为false，其他的都为true，因此，一定要注意[]、{}都会被转为true

3. ?.

if (cur.devices?.length > 0) {}
cur对象的devices数组是否为null，如果不为null，devices数组的长度是否大于0
