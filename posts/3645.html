2.提交表单加loading

4.样式要看下真机预览
5.微信小程序本地图片不能用作css背景，只能用image标签定位
6.不变大小的元素使用px，自适应的使用rpx
7.列表要考虑到没有数据和数据过长的情况

9.参数中出现空格等特殊字符时需要对参数进行编码
编码：encodeURIComponent(JSON.stringify(data))
解码：JSON.parse(decodeURIComponent(data))

10.手机时间修改颜色：”navigationBarTextStyle”: “white”
11.border使用px单位,手机相关设备数据使用px单位
12.使用 uni-app 开发小程序，不管是使用微信小程序的 wx.showToast() API 或 uni-app的 uni.showToast() API 显示消息提示框，显示图标 title 文本最多显示 7 个汉字长度，在不显示图标的情况下，大于两行不显示。可以使用u-view的toast组件
13.微信小程序传参为空就写成空字符串’’,不要写成null和undefined，会变成’null’和’undefined’
14.sass公共样式变量可以定义在 uni.scss 文件中
