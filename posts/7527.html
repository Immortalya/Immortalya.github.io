1.vue包含一组观察数组的变异方法，push()、pop()、shift()、unshift()、splice()、sort()、reverse()，调用这些方法就可以触发视图的更新。

2.vue官网的全局API里的set方法

//直接赋值
this.list[0] = 444
//vue.set 需要修改的数组/对象，索引值/属性，新值
this.$set( this.list, 0, 444 )
- 注：直接赋值不能在view层重新渲染

8.vue3引入 ts文件里的json数据

json数据: utils/languages.ts

vue文件：header.vue

<script lang="ts">
  import lang from '/@/utils/language';

  let langList = [];
  const getLangList = () => {
    langList = [];
    for (let key in lang.languages) {
      langList.push({
        label: lang.languages[key].name,
        value: lang.languages[key].abbreviation,
      });
    }
  };
  getLangList();
</script>
