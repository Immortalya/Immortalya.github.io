1.看板娘-下载单个模型

1.安装依赖

#基础支持的插件
npm install --save hexo-helper-live2d
#单个live2d模型
npm install --save live2d-widget-model-haruto

2._config.yml配置

# Live2D
## https://github.com/EYHN/hexo-helper-live2d
live2d:
  enable: true #开关插件版看板娘
  scriptFrom: local # 默认
  # scriptFrom: https://cdn.cbd.int/live2d-widget@3.x/lib/L2Dwidget.min.js # 你的自定义 url
  tagMode: false # 标签模式, 是否仅替换 live2d tag标签而非插入到所有页面中
  debug: false # 调试, 是否在控制台输出日志
  model:
    use: live2d-widget-model-hijiki # npm-module package name
    # use: https://unpkg.com/live2d-widget-model-koharu@1.0.5/assets/koharu.model.json # 你的自定义 url
  display:
    position: left #控制看板娘位置
    width: 150 #控制看板娘大小
    height: 300 #控制看板娘大小
  mobile:
    show: true # 手机中是否展示

2. 永久链接

一个Hexo插件，用于根据帖子前端的标题和数据生成静态帖子链接。

    npm install hexo-abbrlink --save
    官网： https://github.com/rozbo/hexo-abbrlink
    配置_config.yml
    1. permalink: posts/:abbrlink.html
    2. language: zh-CN
    3. 在文件最后面空白处添加
        # abbrlink config
        abbrlink:
          alg: crc32      #support crc16(default) and crc32
          rep: hex        #support dec(default) and hex

3.插入图片

一个Hexo插件，自动给hexo中的asset image一个绝对路经。

    npm install hexo-asset-image --save
    官网： https://github.com/b40yd/hexo-asset-image
    配置_config.yml
    1. post_asset_folder: true

插入图片

MacGesture2-Publish(文件夹)
├── apppicker.jpg(插入图片)
├── logo.jpg
└── rules.jpg
MacGesture2-Publish.md(文件)

只需要用于![logo](logo.jpg)插入logo.jpg。

4.私密文章

npm install hexo-blog-encrypt --save

配置_config.yml
# 文章加密
encrypt:
  enable: true
  abstract: 这是一篇加密文章，内容可能是个人情感宣泄或者收费技术。如果你非常好奇，请与我联系。
  message: 输入密码，查看文章。

文章Front-matter中加上password: ******即可：
---
title: Hello World
abbrlink: 3eeb
password: ******
---

自定义提示语：

---
title: Hello World
abbrlink: 3eeb
password: 123456
abstract: 密码：123456
message: 看不到吧，hhhh，不告诉你密码是123456
---

password: 是该博客加密使用的密码
abstract: 是该博客的摘要，会显示在博客的列表页
message: 这个是博客查看时，密码输入框上面的描述性文字
