2.图表应用主题

1.引入echarts

npm install echarts --save

2.项目中引入Echarts

main.js

import * as echarts from 'echarts';
Vue.prototype.$echarts = echarts  //定义为全局变量

3.引入官方主题

https://echarts.apache.org/zh/download-theme.html
例如macarons主题

3.1. 引入官方定义的主题macarons 

require("echarts/theme/macarons");

3.2. 初始化echarts实例时引入主题名称

this.chart = echarts.init(document.getElementById("main"), "macarons");

4.引入自定义主题

https://echarts.apache.org/zh/theme-builder.html

4.1. 下载主题
根据所需定义好主题后就可以下载（这里选择了JS版本）
例如自定义命名为first主题

下载或复制以下的主题保存至 first.js 文件；

4.2. 引入first.js文件

import first from "./first";

4.3. 初始化echarts实例时引入主题名称

this.chart = echarts.init(document.getElementById("main"), "first");
