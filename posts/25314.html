1. 扩展运算符

扩展运算符是三个点（…）。它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列。
『…』 扩展运算符能将『数组』转换为逗号分隔的『参数序列』

扩展运算符的应用

2.复制数组

3.与解构赋值结合
扩展运算符可以与解构赋值结合起来，用于生成数组。

注：如果将扩展运算符用于数组赋值，只能放在参数的最后一位，否则会报错。

4.字符串
扩展运算符还可以将字符串转为真正的数组。

5.Map 和 Set 结构，Generator 函数
扩展运算符内部调用的是数据结构的 Iterator 接口，因此只要具有 Iterator 接口的对象，都可以使用扩展运算符，比如 Map 结构。

注：如果对没有 Iterator 接口的对象，使用扩展运算符，将会报错。

6.将伪数组转为真正的数组

2.Array.from()

Array.from()方法用于将两类对象转为真正的数组：类似数组的对象（array-like object）和可遍历（iterable）的对象（包括 ES6 新增的数据结构 Set 和 Map）。

1.Array.from(array, fn, this)

Array.from()方法就是将一个类数组对象或者可遍历对象转换成一个真正的数组，也是ES6的新增方法。
参数：
array：要变换成数组的为数组对象或可迭代对象
fn: 指定了该参数，新数组中的每个元素都会执行该回调函数
this: 执行回调函数时的this对象返回值：一个新的数组实例
1、将类数组对象转换为真正数组：

类似数组：就是含有length和索引属性的对象
const obj = { 0: 'name', 1: 'age', 2: 'sex', 3: 'height' length: 3,}
注意：类似数组对象的length的值，决定了返回数组的长度
可迭代对象：Array、Set、Map和字符串都是可迭代对象
