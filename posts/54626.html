HTML 头部包含 HTML 元素的内容，与 元素内容不同，页面在浏览器加载后它的内容不会在浏览器中显示，它的作用是保存页面的一些元数据。

1 标题

(en-US) 元素在页面加载完毕时显示在页面中，通常只出现一次，用来标记页面内容的标题（故事名称、新闻摘要等等）。 元素是一项元数据，用于表示整个 HTML 文档的标题（而不是文档内容）。 <h3 id="2-元数据-元素"><a href="#2-元数据-元素" class="headerlink" title="2 元数据 元素"></a>2 元数据<meta> <pre><code>name 指定了 meta 元素的类型；说明该元素包含了什么类型的信息。 content 指定了实际的元数据内容。 </code></pre> <p>这两个 meta 元素对于定义你的页面的作者和提供页面的简要描述是很有用的。</p> <p>2.1 <code>name</code>常见属性值</p> <pre><code>author：作者 minimum-scale：允许用户缩放到的最小比例。<br> user-scalable：用户是否可以手动缩放。</p> <p>3.http-equiv</p> <h3 id="3-在站点增加自定义图标"><a href="#3-在站点增加自定义图标" class="headerlink" title="3.在站点增加自定义图标"></a>3.在站点增加自定义图标</h3><p>为了进一步丰富你的网站设计，你可以在元数据中添加对自定义图标的引用，它们会在某些场景下显示。最常见的用例为 favicon（为“favorites icon”的缩写，在浏览器的“收藏夹”及“书签”列表中显示）。</p> <p>页面添加网页图标的方式有：<br>1.将其保存在与网站的索引页面相同的目录中，以 .ico 格式保存（大多数浏览器支持更通用的格式，如 .gif 或 .png）<br>2.将以下行添加到 HTML 的 <head> 块中以引用它：<br><code><link rel="icon" href="favicon.ico" type="image/x-icon" /></code></head></p> <h3 id="4-在HTML中引入CSS"><a href="#4-在HTML中引入CSS" class="headerlink" title="4.在HTML中引入CSS "></a>4.在HTML中引入CSS <link></h3><p>要链接一个外部的样式表，你需要像这样在你的<head>中包含一个<link>元素<br><code><link rel="stylesheet" href="main.css" type="text/css"></code><br>href 属性设置外部资源的路径<br>rel 属性值定义了所链接的资源与当前文档的关系，属性的值为“stylesheet”(样式表)。<br>type 这个属性被用于定义链接的内容的类型。这个属性的值应该是像 text/html，text/css 等 MIME 类型。</head></p> <p>rel 属性定义了所链接的资源与当前文档的关系，在 <a>、<area> 和 <link> 元素上有效。支持的值取决于拥有该属性的元素。</a></p> <h3 id="5-在HTML中引入js"><a href="#5-在HTML中引入js" class="headerlink" title="5.在HTML中引入js "></a>5.在HTML中引入js <script></h3><p>使用 script 元素来导入（外部）脚本。<br><code><script src="javascript.js" type="text/javascript"></script></code><br>src 属性定义引用外部脚本的 URI<br>type 这个属性被用于定义链接的内容的类型。</p> <h3 id="6-为文档设定主语言"><a href="#6-为文档设定主语言" class="headerlink" title="6.为文档设定主语言"></a>6.为文档设定主语言</h3><p>为站点设定语言，这个可以通过添加 lang 属性到 HTML 开始的标签中来实现<br><code><html lang="zh-CN"></html></code></p> <p>en：英语<br>zh-CN：中文</p> </script></h3>
